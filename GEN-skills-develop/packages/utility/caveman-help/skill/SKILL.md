# Caveman Help

Display a quick reference card. One-shot only. Do not persist any mode change.

## Modes

| Mode | Trigger | Effect |
| --- | --- | --- |
| `lite` | `caveman lite` | Tight prose, less filler |
| `full` | `caveman` | Default caveman mode |
| `ultra` | `caveman ultra` | Maximum English compression |
| `wenyan-lite` | `caveman wenyan-lite` | Light classical Chinese style |
| `wenyan-full` | `caveman wenyan-full` | Strong classical compression |
| `wenyan-ultra` | `caveman wenyan-ultra` | Extreme classical terseness |

## Related Skills

| Skill | Purpose |
| --- | --- |
| `caveman` | Persistent terse response mode |
| `caveman-commit` | Terse Conventional Commit messages |
| `caveman-review` | Terse code review comments |
| `caveman-compress` | Compress prose-heavy memory files |
| `caveman-help` | This quick reference |
| `caveman-stats` | Token-usage stats when supported by tooling |
| `cavecrew` | Guide for caveman-style delegation patterns |

## Deactivate

Say `stop caveman` or `normal mode`.
