# Office Kit

Use this skill to route office document tasks to the focused document skills installed with it.

- Use `docx` for Word document creation, editing, review, comments, tracked changes, formatting preservation, and DOCX analysis.
- Use `pdf` for PDF extraction, generation, splitting, merging, form inspection, form filling, and document analysis.
- Use `pptx` for PowerPoint creation, editing, analysis, conversion, slide assets, speaker notes, comments, and raw OOXML presentation work.
- Use `xlsx` for spreadsheet creation, editing, formula work, recalculation, workbook validation, tabular analysis, and CSV/TSV workbook workflows.
- Use `markitdown` for local document-to-Markdown conversion with Microsoft MarkItDown.

Prefer the narrowest matching skill. If a request spans multiple office formats, use the relevant leaf skills together and keep intermediate files package-local to the current task workspace.
